# 🍕 Sergio & Robi’s Pizzeria Website

A modern, responsive website designed and developed for a premium Italian pizza catering business based in Northampton, UK.

This project focuses on clean design, strong visual identity and conversion-driven layout to attract private event bookings such as weddings, parties and corporate events.

---

## 🌐 Live Website

👉 https://sergiorobipizzeria.co.uk

---

## ✨ Project Overview

This website was created to showcase a mobile wood-fired pizza catering service and help convert visitors into enquiries and bookings.

The design combines:

* a premium dark aesthetic
* Italian-inspired colour palette
* modern layout with clear structure
* strong call-to-action elements

---

## 🚀 Key Features

* 📱 Fully responsive design (mobile-first approach)
* 🍕 Event-focused layout (weddings, private parties, corporate events)
* 🔥 Hero section with strong value proposition and CTA
* 💬 WhatsApp integration for quick enquiries
* ⭐ Testimonials section to build trust
* 📍 “Find Us” page with weekly locations and Google Maps
* 📋 Full menu page with structured content
* 🌍 Bilingual version (English / Italian)
* ⚡ Smooth animations using IntersectionObserver
* 🎯 Conversion-focused UX design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (custom design system with variables)
* Vanilla JavaScript
* Google Fonts (Inter & Playfair Display)

---

## 🎨 Design Approach

The goal was to create a website that feels:

* premium but accessible
* authentic and Italian-inspired
* clean, modern and easy to navigate

Special attention was given to:

* typography hierarchy
* spacing and layout balance
* subtle animations and hover effects
* clear user journey → from landing to booking

---

## 📈 Business Goal

The primary objective of this website is to:
👉 generate enquiries for private event catering

Key UX decisions were made to:

* highlight services clearly
* build trust quickly
* encourage immediate contact (Call / WhatsApp / Enquiry form)
